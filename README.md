# 🚀 GenAI-Powered Object Detection System

An AI-powered object detection application built using modern **Generative AI and deep learning frameworks**, capable of identifying and localizing multiple objects in an image.

---

## 🔍 Overview

Object detection is a computer vision task that identifies objects in an image and outputs bounding boxes with labels. ([Hugging Face][1])

This project demonstrates the practical use of **Generative AI ecosystems (Hugging Face)** combined with computer vision techniques using pre-trained models.

---

## ⚙️ Features

* Multi-object detection with bounding boxes
* Confidence score-based predictions
* Interactive web app (Hugging Face Spaces)
* Fast inference using pre-trained models

---

## 🧠 GenAI Skills Demonstrated

* Foundation model usage
